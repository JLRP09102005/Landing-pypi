# Python.org Website Clone - Práctica de HTML y CSS

## 📝 Descripción

Este proyecto es una **réplica educativa** del sitio web oficial de Python (python.org), creado con fines de aprendizaje y práctica de tecnologías web front-end.

> ⚠️ **Aviso importante**: Este es un proyecto de práctica personal. No está afiliado, asociado, autorizado ni respaldado por la Python Software Foundation. Todos los derechos de diseño y marca registrada pertenecen a sus respectivos propietarios.

## 🎯 Objetivo del Proyecto

El propósito de este proyecto es practicar y mejorar mis habilidades en:

- **HTML5** semántico y estructurado
- **CSS3** moderno (Flexbox, Grid, animaciones)
- Diseño responsive y mobile-first
- Buenas prácticas de desarrollo web
- Estructura y organización de código

## 🛠️ Tecnologías Utilizadas

- HTML5
- CSS3
